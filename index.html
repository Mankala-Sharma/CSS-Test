Absolute and relative Positioning

Absolute Position is another CSS technique that adjusts an element’s position relative to its parent. If no parent element is present, the document body is used as the parent. The syntax for absolute positioning is position: absolute;.

Relative positioning is a CSS technique that allows an element to be adjusted from its normal position. The syntax for relative positioning is position: relative;.
When we set the top, right, bottom, and left properties of an element with relative positioning, it moves from its original location. However, it’s important to note that other elements on the page will not fill the gap left by the moved element.

z-index

The z-index property specifies the stack order of an element.It is used to define the order of elements if they overlap on each other.
An element with greater stack order is always in front of an element with a lower stack order.

Padding and Margin

Margin is the space around an element.It is used to move an element up or down on a page as well as left or right.

Padding is the space between the element and the related content inside it. It determines how elements look and sit within a container.
